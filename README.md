# 放射線・核物質 規制経路ナビゲーター

RN事案の擬律の入口を可視化する単一HTMLツール。
物質×行為から、関係法令・委任経路・照会メモまでを組み立てる。

- 現行版：v0.2（性状レイヤー追加）／データ基準日 2026-07-05
- 本体：`index.html`（依存ライブラリなし・オフライン動作）
- 更新手順：`index.html` 内の LAWDATA を編集して main へ push すると自動で再デプロイされる
- 検証運用：各項目は e-Gov 原文照合を経て status を「検証済み」へ更新する（詳細は本体フッター参照）

## 注意

本リポジトリは公開である。公開法令の対応表・ツール本体のみを置き、
事案情報・内部情報・個人情報は一切置かないこと。
